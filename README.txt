A program that copies data from an imported excel file and generates as many files as suppliers. This is used to report some data.
HOW TO USE:
	1. Select "Import file"
	2. Select "Save file to..."
	3. Press Generate files.
ADD NEW SUPPLIER TO DATABASE:
	1. Click onn "Add new supplier".
	2. Add the KEY (Supplier Network User Code - 6 uppercase letters).
	3. Add the VALUE (Supplier Name).
	4. Click on "Add New Supplier".
DELETE SUPPLIER:
	1. Click on "Delete supplier" button.
	2. Enter the supplier Newtork User Code(example: "EGROMS" - 6 letters).
	3. Hit "Delete supplier".
